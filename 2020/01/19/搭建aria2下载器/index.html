搭建aria2下载器

发表于 2020-01-19 | 阅读次数:
字数统计: 457 | 阅读时长 ≈ 2

基于win10搭建

aria2是一个没有GUI的服务器，使用它下载的文件不会被限速

下载aria2

创建一个新的目录把下载好的文件解压进去

创建文件

  1. Aria2.log // 日志文件

  2. aria2.session // 下载历史

  3. aria2.conf // 配置文件

  4. HideRun.vbs // 隐藏cmd窗口

设置配置文件

aria2.conf写入

写法 功能
dir 下载文件保存路径
log 日志文件，不需要可以删掉这行
input-file 与 save-session 记录和读取下载历史用的，断电和重启时保证下载任务不会丢失，如果有时aria2不能启动，清空这里面的内容就行了
