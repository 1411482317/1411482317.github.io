指针与引用

指针指向内存的地址值

int *p; //为创建一个指针

*p; //解引一个指针，获取指针内的值，假如指针没有值，则会出现未定义的操作

p++; //p指向p的下一个地址值

p + 10; //p指向了p的后10个位置的地址值

p–; //p指向了p之前1位的地址值

p - 10; //p指向了p之前10位的地址值

p == p2; //地址值一样则true

p != p2 //地址值不一样则true

操作空指针的风险非常大！

int a = 2;

//注意：在=之前的&叫做取别名，在=之后的&叫做取地址值

int &b = a; //&b为a的别名，修改b的值，a的值也会被修改

int *p2 = &a; //指针p2指向了a的地址值

*p2 = 33; //解引指针获取指针内的值，修改值，其值改为33，此时a的值为33

注意：数组的名称是一个指针，它指向了数组里的第一个元素
